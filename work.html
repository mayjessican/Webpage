MyFlix Movie App

Hosted on Heroku, I created a REST API with Node.js to build a server-side component of a "movies" web application. This app gave users access to information about different movies, genres, and directors. Users can sign up (register and deregister), update their personal information, and create their own list of favourite movies (as well as delete them).

Technolgy used:

- JavaScript

- MongoDB

- MongoDB Atlas

- React

- Redux

- REST API

- Node.JS

- Bootstrap

- Mongoose

Meet App

Here I created a serverless, progressive web application using a test driven development technique that uses the Google Calendar API to fetch upcoming events. This app allows users to filter events by city, show/hide event details, specify the number of events, use the app when offline, add an app shortcut to the home screen, and view a chart showing the number of upcoming events by city.

Technology used:

- React

- CSS

- Puppeteer

- Jest

- Cucumber

- Enzyme

Mobile Chat App

React Native App built for IOS and android devices that allowes users to choose their username and background colour, send/recieve messages, see messages offline, share their location, send/receive photos from their mobile library, and send a photo direct from their camera. This app uses firebase/firestore to store any messages, images, etc. and also is compatible with a screen reader. I used a Kanban board to track my progess and please see the link to access it!

Technology used:

- React Native

- Expo

- Google firestore database

- Gifted Chat Library

- CSS

- JSdoc
